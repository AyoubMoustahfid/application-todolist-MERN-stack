# Application todolist MERN stack

## API Reference

#### Get all task

```http
  GET /api/task/all
```

#### Post:  create Task

```http
  POST /api/task/create
```


#### PUT: update Task

```http
  PUT: /api/task/update/:taskId
```


#### DELETE: delete Task

```http
  DELETE /api/task/delete/:taskId
```
